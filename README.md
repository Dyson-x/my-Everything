 ## 这是一个很不错的Everything的纯Java版工具

---

### 1.关于Everyhing 文件搜索工具

**仿照Everything桌面工具，基于纯Java语言开发的命令行搜索工具**

### 2.背景

**为了解决Windows下文件查询响应过慢问题，自主实现了一款命令行搜索工具**

### 3.优点

- **界面简洁易用**

- **通过建立索引，实现快速搜索**
- **体积小，占用很低的系统资源**
- **实时跟踪文件变化**
